## 1. 设计目标与原则

### 1.1 目标感受

> **像在读一本编排精良的纸质圣经，而不是在刷信息流。**

用户应感到：安静、清晰、不累眼、层次一眼能分辨（叙述 / 对话 / 引用 / 专名），需要时能一点即懂。

### 1.2 五条原则

| 原则 | 说明 |
|------|------|
| **经文第一** | UI Chrome 退后；字号、对比度、行距优先于装饰 |
| **层次靠排版，不靠花哨** | 对话、引用、专名用字重/缩进/符号区分，少用高饱和色块 |
| **默认可读，专家可深** | 默认干净；专名下划线、脚注、原文等可设置开关 |
| **中日混排友好** | 新译本以中文为主，夹英文专名、KJV 对照时不乱行 |
| **点读不打断** | 点击专名/脚注用轻量浮层，不全屏跳转 |

### 1.3 参考标杆

- **微读圣经**：纸质感、简洁、节号克制  
- **Kindle / Apple Books**：行距、边距、主题切换  
- **经典纸质圣经排版**：卷首标题、诗歌体、对话引号传统

## 2. 版式总览

### 2.1 阅读区域栅格

```
┌────────────────────────────────────────┐
│ 安全区顶栏（章标题、译本、设置）  56pt   │
├────────────────────────────────────────┤
│ ← 边距 20pt →                          │
│                                        │
│   16  神爱世人，甚至将他的              │
│   独生子赐给他们…                      │
│                                        │
│   17  因为神差他的儿子降世…            │
│                                        │
│ ← 边距 20pt →                          │
├────────────────────────────────────────┤
│ 底栏工具区（可收起）            48–56pt │
└────────────────────────────────────────┘
```

| 参数 | 手机竖屏 | 平板 / 大屏 |
|------|----------|-------------|
| 左右边距 | 20–24 pt | 32–48 pt（最大阅读宽 680 pt 居中） |
| 顶栏高度 | 48–56 pt | 同左 |
| 底栏 | 48 pt（收起时仅 8 pt 指示条） | 同左 |
| 最大行宽 | 全宽减边距 | **34–38 个汉字/行** 为舒适上限 |

**长行是阅读疲劳主因。** 平板务必限制最大行宽并居中，不要拉满全屏。

### 2.2 垂直节奏

- 节与节之间：`0.6em` 额外间距（相对正文字号）
- 段落之间：`1.2em`
- 诗歌体每行：`0.35em`（更紧凑，见 §7.3）

## 3. 背景与主题

### 3.1 四套主题（MVP 必做）

| 主题 | 背景 | 正文 | 适用场景 |
|------|------|------|----------|
| **浅色**（默认） | `#FAFAF8` 暖白 | `#1A1A1A` | 日间一般环境 |
| **护眼** | `#F3E9D6` 羊皮纸 | `#2C2416` 深褐 | 长时间阅读、暖光环境 |
| **深色** | `#121212` | `#E8E6E3` 略暖灰白 | 夜间 |
| **纯黑 AMOLED** | `#000000` | `#E0E0E0` | OLED 省电夜间（可选） |

**避免：** 纯白 `#FFFFFF` 刺眼；纯黑底 + 纯白字对比过激。

### 3.2 背景质感（可选增强）

- 浅色/护眼：极轻纸张噪点（opacity 2–3%），无纹理时亦可用
- 深色：无噪点，保持干净
- **禁止** 复杂插图背景干扰阅读

### 3.3 主题与系统联动

- 默认：跟随系统
- 可选手动锁定某一主题
- 切换主题：**无动画或 200ms 淡入**，避免闪白

## 4. 字体系统

### 4.1 字体角色

| 角色 | 中文 | 英文/对照 | 用途 |
|------|------|-----------|------|
| **经文正文** | 思源宋体 / 方正书宋 / 系统「宋体」类 | — | 新译本正文 |
| **UI 界面** | 思源黑体 / SF Pro / 系统无衬线 | 同左 | 顶栏、按钮、浮层 |
| **节号** | 与 UI 同族无衬线 | — | 小字、tabular |
| **KJV 对照** | — | **Literata** / **Charter** / **New York** / Georgia | 英文衬线，与中文协调 |
| **专名音译** | 正文宋体 | 小型无衬线仅用于 Strong's 编号 | 原文浮层 |

### 4.2 字体选择理由

- **经文用衬线（宋体类）**：中文圣经传统排版习惯；笔画清晰，长时间阅读优于黑体
- **UI 用无衬线**：与经文区形成「内容 / 工具」层次
- **KJV 用英文衬线**：圣经英文排版传统，与新译本宋体在气质上匹配

### 4.3 字重

| 元素 | 字重 |
|------|------|
| 正文 | Regular (400) |
| 对话（可选强调） | Regular，靠引号与缩进区分，不加粗 |
| 卷/章标题 | Medium (500) |
| 专名（可点击） | Regular + 下划线/底纹，不加粗 |
| 节号 | Regular 或 Medium，颜色弱化 |

### 4.4 字体许可

- 优先使用可嵌入的商业免费字体（思源系列）或系统字体 fallback
- iOS：PingFang + Songti SC；Android：Noto Serif CJK SC

## 5. 字号与行距

### 5.1 字号阶梯（以 pt 为单位，支持系统动态类型缩放）

| 档位 | 经文字号 | 节号 | KJV 对照 | 行高 |
|------|----------|------|----------|------|
| 小 | 16 | 11 | 14 | 1.65 |
| **标准**（默认） | **18** | **12** | **15** | **1.75** |
| 大 | 20 | 13 | 16 | 1.80 |
| 特大 | 22 | 14 | 17 | 1.85 |
| 巨大 | 24 | 15 | 18 | 1.90 |

- 默认 **18pt / 行高 1.75**：约每屏 7–9 节（视章节而定）
- 节号字号 = 正文的 **66–70%**，绝不与正文同大

### 5.2 行距与字距

| 参数 | 值 |
|------|-----|
| 行高 line-height | 1.75（标准档） |
| 字距 letter-spacing | 中文 0.02em（略疏，提升识别） |
| 段落间距 | 1.2em |
| 两端对齐 | **关闭**（中文圣经使用左对齐） |

### 5.3 用户调节

阅读设置 `Aa` 面板：

```
字号  ────●─────  18pt
行距  ───●──────  标准
字距  ──●───────  默认（极少人调，可藏进阶里）
```

调节时正文区 **实时预览**，无需确认按钮。

## 7. 段落与结构

### 7.1 叙述体（默认）

新译本大部分为散文叙述，按 **语义段落** 分组，而非每节一段。

```
　　耶稣回答说：「我实实在在地告诉你，若不重生，就不能见神的国。」

16　神爱世人，甚至将他的独生子赐给他们，叫一切信他的，不致灭亡，反得永生。

17　因为神差他的儿子降世，不是要定世人的罪，乃是要叫世人因他得救。
```

规则：

- 段首 **空两格**（全角空格）或 `1.5em` 首行缩进，二选一全局统一
- **同一对话块** 内多节可合并为一段（见 §9）
- 每节仍保留节号，节号在段首或行首

### 7.2 节号位置（两种模式，用户可选）

| 模式 | 样式 | 适合 |
|------|------|------|
| **行内上标**（默认） | <sup>16</sup> 神爱世人… | 紧凑、传统印经感 |
| **行外悬挂** | 节号在左边距外列，正文对齐 | 节号极清晰、适合查考 |

```
行外悬挂示意：

14  │  神爱世人，甚至将他的
    │  独生子赐给他们…
15  │  叫一切信他的…
```

**推荐默认：行内上标**，查考模式可切换行外悬挂。

### 7.3 诗歌体（诗篇、赞美诗等）

识别 `poetry` 类型章节时自动切换：

```
　　诸 heavens 述说神的荣耀；  ← 实际为中文经文，仅示意结构
　　穹苍传扬他的手段。
　　天天发出言语，
　　夜夜传扬知识。
```

| 规则 | 值 |
|------|-----|
| 行高 | 1.6（略紧于散文） |
| 平行行 | 左对齐，第二行起缩进 `2em`（希伯来诗歌「阶梯体」） |
| 节号 | 每逻辑行首或每对行首 |
| 段间距 | 诗段之间 `1.5em` |

### 7.4 家谱/清单体（如马太福音 1 章）

```
　　亚伯拉罕的后裔，大卫的子孙，耶稣基督的家谱：
　　亚伯拉罕生以撒；以撒生雅各；雅各生犹大和他的弟兄；…
```

- 分号 `；` 后可选轻换行（设置项：「清单换行」开/关）
- 默认 **不换行**，保持连续；开启后每 `；` 换行并缩进

## 8. 节号与分节

### 8.1 节号样式

```css
/* 逻辑规格 */
verse-number {
  font-family: UI 无衬线;
  font-size: 0.68em;
  color: secondary;
  font-variant-numeric: tabular-nums;
  margin-right: 0.35em;
  vertical-align: super; /* 上标模式 */
  user-select: none;     /* 复制经文时可配置是否包含节号 */
}
```

### 8.2 章标题

```
┌─────────────────────────────┐
│         约翰福音              │  ← 卷名（顶栏或章首）
│      第 3 章                  │  ← 章号居中，字号 1.3× 正文
│  ─────────────────────       │  ← 极细分割线 optional
└─────────────────────────────┘
```

- 卷名：顶栏显示为主，章首可重复一次
- 章号：**Medium**，字号 = 正文 × 1.25，上下留白 `1.5em`

### 8.3 小标题（section headings）

部分译本带段落标题（如「尼哥底母问重生」），若有数据：

- 字号 = 正文 × 0.95
- 颜色 = 次要色，**斜体不用**（中文斜体观感差）
- 上下 `1em` 留白
- 可点击折叠本章大纲（进阶）

## 9. 对话排版

### 9.1 对话识别

经文中以下模式标记为 `dialogue`（数据层或规则推断）：

- 中文引号 `「」`、`『』` 包裹
- 「说：」「回答说：」「问说：」等动词 + 引文
- 连续对话节合并为同一段落块

### 9.2 对话视觉（推荐方案）

**方案 A：引号 + 段首缩进（默认）**

```
　　耶稣回答说：「我实实在在地告诉你，若不重生，就不能见神的国。」

　　尼哥底母说：「人已经老了，岂能重生进去吗？」
```

**方案 B：左侧对话线（查考模式可选）**

```
│  耶稣回答说：「我实实在在地告诉你…」

│  尼哥底母说：「人已经老了…」
```

对话线颜色：`#D4CFC4`（浅）/ `#3A3835`（深），宽 2pt，左边距内缩。

### 9.3 说话者标注

| 类型 | 样式 |
|------|------|
| 「某某说：」 | 正文色，「说：」三字用次要色 |
| 「耶稣说：」 | 「耶稣」走专名规则（可点击）；动词「说：」次要色 |
| 无引导直接引语 | 仅引号，不额外标记 |

### 9.4 对话符号规范

| 符号 | 用法 |
|------|------|
| `「」` | 第一层引语（默认） |
| `『』` | 引语内的引语（嵌套） |
| `……` | 省略，与正文同色 |
| `——` | 对话中断、话题转换，单独成行时上下留白 |

**不在引号外再用颜色区分对话**，避免彩虹感。

### 9.5 长篇讲论（如登山宝训）

- 同说话者连续多节：**合并段落**，节号保留在各节行首
- 每 5–8 节可插入 **不可见软换行提示**（滚动时自然断）
- 章首显示「耶稣说：」一次，后续节若无新说话者可省略重复「耶稣说」

## 10. 引用与嵌套文字

### 10.1 引用类型

| 类型 | 示例 | 样式 |
|------|------|------|
| 经内对话 | 「…」 | §9 |
| 旧约引文 | 先知书引摩西律法 | 引用色 `#2C4A6E` + 左侧 2pt 竖线 |
| 书信引诗 | 保罗引诗篇 | 同上 + 可选缩进 `1em` |
| 叠字强调 | 「世世生生」 | 保持正文，不特殊处理 |

### 10.2 旧约引文样式

```
　　经上记着说：│ 我要差遣我的使者在你前面，
              │ 预备道路。
```

- 竖线颜色：引用色，opacity 60%
- 「经上记着说」等引导语：次要色
- 点击引文 → 跳转被引旧约章节（若有交叉引用数据）

### 10.3 用户高亮与引用色区分

用户高亮 **绝不** 使用与「圣经引文」相同的蓝色，避免混淆。引文 = 固定语义色；高亮 = 用户自选色。

## 11. 专名标注（人名/地名/称谓）

### 11.1 专名分类

| 类型 | 示例 | 默认样式 |
|------|------|----------|
| 人名 | 耶稣、尼哥底母、摩西 | 可点击 + 淡底纹或点下划线 |
| 地名 | 耶路撒冷、加利利 | 同左 |
| 称谓/头衔 | 主、基督、人子、法利赛人 | 可点击，样式略弱于人名 |
| 群体 | 以色列人、门徒 | 可选点击，默认与正文同（减少噪音） |

### 11.2 视觉（默认克制）

**推荐：点状下划线 + 极淡底纹（二选一，设置里切换）**

```
神爱世人，甚至将他的独生子赐给他们
         ────
         （「独生子」若标注为神学专有词，可选浅灰下划线，不强制）
```

| 样式 | 规格 |
|------|------|
| 点下划线 | `1pt dotted`，色 `#B8A88A`，offset 2pt |
| 淡底纹 | 圆角 2pt，`#E8E4DC`，padding 0 1pt |
| 深色主题 | 下划线 `#6B6358`；底纹 `#2A2824` |

**默认仅标注「人名 + 地名」**；称谓类在「专名详细标注」开关打开后显示。

### 11.3 密度控制

- 单节专名 ≤ 3 个：正常标注
- 单节专名 > 5 个（如家谱）：**关闭下划线**，仅长按查询，避免「满地波浪线」

## 12. 点击解释交互

### 12.1 点击专名/术语

**轻量浮层（Popover）**，非全屏，不遮挡整节：

```
┌─ 尼哥底母 ──────────────────┐
│ 法利赛人中的官，以色列人的官 │  ← 1–2 句静态释义
│ [📖 词典]  [✨ 问助手]  [×]  │
└─────────────────────────────┘
```

| 区块 | 内容 |
|------|------|
| 标题 | 专名 + 类型标签（人名/地名） |
| 摘要 | 静态词典 1–2 句（离线） |
| 操作 | 「词典详情」展开；「问助手」带锚点跳转 AI |

### 12.2 浮层规格

| 参数 | 值 |
|------|-----|
| 最大宽 | 屏幕宽 − 48pt |
| 最大高 | 40% 屏幕高，超出可滚动 |
| 出现动画 | 150ms fade + 8pt 上移 |
| 消失 | 点击外部 / 再次点击专名 / 下滑 |
| 位置 | 优先锚定在专名下方；靠底时翻转到上方 |

### 12.3 解释内容优先级

1. **静态词典**（人名地名，<100ms）
2. **资源指南片段**（注释中关于该专名的段落）
3. **AI 原文/解释模式**（用户点「问助手」才调用）

**绝不** 在点击专名时自动调 LLM（慢、费、打断阅读）。

### 12.4 长按 vs 点击

| 手势 | 行为 |
|------|------|
| **点击** 专名 | 浮层释义 |
| **长按** 经文 | 高亮 / 笔记 / 复制 / 问 AI |
| **双击** | 不使用（易误触） |

---

## 13. 标题与副标题

### 13.1 层级

| 级别 | 元素 | 字号比 | 位置 |
|------|------|--------|------|
| H0 | 卷名 | 1.0×（顶栏） | 顶栏居中 |
| H1 | 章号 | 1.25× | 章首居中 |
| H2 | 段落小标题 | 0.95× | 段前 |
| Body | 经文 | 1.0× | — |
| Caption | 节号、脚注 | 0.68× | — |

### 13.2 卷首书（如诗篇 1）

篇题单独成行，居中，上下 `2em` 留白，颜色次要，不进入正文流。

---

## 14. 脚注与交叉引用

### 14.1 脚注标记

新译本若含脚注（译本差异、字义说明）：

- 用上标 `*` `a` `1` 小字，次要色
- 点击 → 底栏半屏或浮层显示脚注正文
- 脚注正文字号 = 15pt（标准档下），行高 1.6

### 14.2 文内交叉引用链接

「就如…所说」类可点击跳转：

- 样式：引用色 + 虚线下划线
- 点击 → 跳转目标经节，**带回退栈**（返回原位置）

## 15. 对照模式排版

### 15.1 逐节对照（默认）

```
16  神爱世人，甚至将他的独生子赐给他们，
    叫一切信他的，不致灭亡，反得永生。

    For God so loved the world, that he gave
    his only begotten Son, that whosoever
    believeth in him should not perish…
```

| 规则 | 值 |
|------|-----|
| 中文 | 上，宋体 18pt |
| KJV | 下，英文衬线 15pt，次要色 `#6B6B6B` |
| 间距 | 中英文之间 `0.5em` |
| 节号 | 仅出现一次，在行首，管中英文两块 |

### 15.2 分栏对照（横屏 / 平板）

```
│ 16 神爱世人…      │ 16 For God so loved… │
```

- 左栏中文右栏英文，中间 1px 分隔线
- 行高同步（以较高一侧为准）

### 15.3 对照模式下的专名

- 中文专名可点击
- KJV 中对应词（如 `only begotten Son`）可点击 → 对照模式 AI 或原文浮层

## 16. 动效与阅读行为

### 16.1 滚动

- **惯性滚动**，无分页吸附（默认连续滚动）
- 可选「按章分页」：左右滑切换章，适合睡前
- 滚动时顶栏：**渐隐或收缩**（延迟 300ms 后隐藏，下滑显示）

### 16.2 跳转与定位

- 从搜索/计划/交叉引用跳入：**目标节 brief flash**（背景 `#FFF3B0` 1.2s 淡出）
- 不自动弹 AI、不自动展开脚注

### 16.3 加载

- 章节内容本地 SQLite，**无骨架屏**；切换章 < 50ms
- 资源指南/词典异步加载，占位用 2 行灰色 shimmer，不用全屏 loading

### 16.4 底栏工具

- 默认 **收起** 为细条；上滑或点击展开
- 阅读 3 秒无操作后自动收起，扩大阅读区

## 17. 无障碍

| 项 | 要求 |
|----|------|
| 动态类型 | 支持系统字号 ±30%，布局不截断 |
| VoiceOver | 节号读作「第 16 节」；专名读作「尼哥底母，人名，可查看解释」 |
| 对比度 | 见 §6.3 |
| 色盲 | 高亮不用纯红绿区分；用形状/图标辅助 |
| 减少动效 | 尊重系统「减少动态效果」 |

## 18. 数据标记规范（工程）

经文数据除 `book, chapter, verse, text` 外，建议扩展：

```json
{
  "verse": 16,
  "text": "神爱世人，甚至将他的独生子赐给他们…",
  "paragraph_id": "jn3_p2",
  "content_type": "prose",
  "speaker": null,
  "entities": [
    { "start": 0, "end": 2, "text": "神", "type": "divine", "clickable": false },
    { "start": 12, "end": 15, "text": "独生子", "type": "theological", "clickable": true }
  ],
  "quotes": [
    { "start": 0, "end": 28, "level": 1, "quote_type": "dialogue" }
  ],
  "cross_refs": ["Rom.5.8"],
  "poetry_line": null
}
```

| 字段 | 用途 |
|------|------|
| `paragraph_id` | 段落合并、对话块 |
| `content_type` | `prose` / `poetry` / `list` |
| `speaker` | 对话说话者 |
| `entities` | 专名点击、浮层 |
| `quotes` | 引号嵌套、引用色 |
| `poetry_line` | 诗歌阶梯缩进 |

**MVP 可分阶段：** 先 `entities` 人名地名；对话规则用引号解析；诗歌用章节元数据表。

## 19. 验收清单

### 视觉

- [ ] 默认 18pt / 行高 1.75，连续阅读 30 分钟测试者无不适反馈
- [ ] 四套主题切换无闪烁
- [ ] 节号清晰但不抢眼，复制行为符合设置
- [ ] 平板最大行宽 ≤ 680pt

### 结构

- [ ] 对话 `「」` 排版正确，嵌套 `『』` 正确
- [ ] 诗篇等诗歌体有缩进阶梯
- [ ] 旧约引文有区分样式

### 交互

- [ ] 点击专名 < 100ms 出浮层（静态词典）
- [ ] 长按经文出操作菜单，不与专名点击冲突
- [ ] 跳转经节有高亮定位动画
- [ ] 对照模式中英文对齐可读

### 性能

- [ ] 切章 < 50ms（本地）
- [ ] 滚动 60fps，1000 节长章无卡顿

### 无障碍

- [ ] 系统大字号下不破版
- [ ] VoiceOver 经节朗读顺序正确
